THE HANGMAN GAME

Guess the word:

Your guesses:

Guesses Left: 10

Wins: 0

Losses: 0

Computer chooses a random word from an array of words. function that hides a word by replacing the letters with underscores. a guess function that removes a letter from the alphabet array User presses a key to start. Each key press counts down the number of remaining guesses. Keys pressed get displayed. If a key pressed is in the word, then display it. If a user spells out an entire word, they win. Add a point to the win score. If the number of guesses runs out, the player loses. Game resets. Game start.
